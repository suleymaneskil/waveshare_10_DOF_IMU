# waveshare_10_DOF_IMU
Waveshare 10-DOF IMU MPU9255 + BMP180 Sensor Arduino Library and Example Code
Gtihub BMP180 Library: https://github.com/sparkfun/BMP180_Breakout_Arduino_Library/tree/master/src 
Arduino Uno and Waveshare 10-DOF IMU connection:                                                       
  5V ============= VCC                                                                           
  GND============ GND                                                                           
  A4 ============= SDA                                                                            
  A5 ============= SCL
